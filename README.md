# 🌐 Personal Portfolio

Este es mi **portfolio personal**, una aplicación web desarrollada con **Next.js** y **TailwindCSS** donde muestro mi experiencia, proyectos y habilidades como **desarrollador web**.

## ✨ Estructura

- **Parte pública (portfolio):**  
  Incluye las secciones principales visibles en la web:  
  - **Home / Hero** → presentación inicial.  
  - **About me** → breve descripción personal.  
  - **Projects** → trabajos y proyectos destacados.  
  - **Skills** → tecnologías y herramientas que domino.  
  - **Contact** → redes y formulario de contacto.  

- **Parte privada (admin):**  
  Panel de administración al que solo yo tengo acceso. Desde aquí puedo:  
  - Gestionar proyectos.  
  - Actualizar habilidades.  
  - Editar la sección *About me*.  
  - Revisar mensajes enviados desde el formulario de contacto.  
  - Configurar opciones generales del portfolio.  

- **Área de usuario (opcional):**  
  Pensada para usuarios autenticados que envíen mensajes, con funcionalidades mínimas:  
  - Cambiar contraseña.  
  - (Opcional) Consultar el historial de mensajes enviados.  

## 🚀 Tecnologías principales

- [Next.js](https://nextjs.org/)  
- [React](https://react.dev/)  
- [TailwindCSS](https://tailwindcss.com/)  
- [Prisma](https://www.prisma.io/) (para base de datos)  
- [Next Auth](https://next-auth.js.org/) (para autenticación)
